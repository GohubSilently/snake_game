## Описание
Классическая игра **«Змейка»**, реализованная на библиотеке **Pygame**.

---

## Игровая механика
- Змейка движется по клеточной сетке фиксированного размера.
- Направление движения изменяется клавишами стрелок.
- Змейка не может мгновенно развернуться в противоположную сторону.
- Игровое поле зациклено: выход за границу переносит змейку на противоположную сторону экрана.
- Яблоко появляется случайно в свободной клетке поля.
- При поедании яблока длина змейки увеличивается.
- Если змейка сталкивается с собственным телом — происходит сброс позиции и длины.
- Текущая длина и рекорд отображаются в заголовке окна игры.

## Запуск проекта локально
1. Клонируем репозиторий.
```bash
git clone https://github.com/GohubSilently/snake_game && cd snake_game
```

2. Установливаем зависимости.
```bash
python3 -m venv .venv && source .venv/bin/activate
pip install --upgrade pip && pip install -r requirements.txt
```

3. Запускаем проект.
```bash
python snake.py
```

4. Проверяем тесты и стилистику.
```
flake8 constants.py && flake8 snake.py
pytest
```

## Управление

| Клавиша     | Действие        |
|-------------|-----------------|
| ↑           | Движение вверх  |
| ↓           | Движение вниз   |
| ←           | Движение влево  |
| →           | Движение вправо |
| ESC         | Выход из игры   |
